# AI-Cyber-Spam-Shield
An AI-based spam detection system that uses Natural Language Processing and Machine Learning (Naive Bayes) to classify SMS/messages as spam or safe, featuring an interactive Streamlit web app with visual analytics and explainable AI outputs.

---

## 🚀 Features
- Spam vs Safe message detection
- Machine Learning (Naive Bayes)
- NLP using CountVectorizer
- Streamlit web app
- Probability & risk visualization
- Feature explanation (length, caps, links)

---

## 🧠 Tech Stack
Python, Streamlit, Pandas, NumPy, Scikit-learn, Matplotlib

---

## ⚙️ Run Project

pip install streamlit pandas numpy scikit-learn matplotlib
python -m streamlit run main.py

📌 Example

Spam: “Congratulations! You won 1,00,000 Rs click now”
Safe: “Hey, are you coming to college today?”
